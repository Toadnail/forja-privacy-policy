Política de Privacidade – Forja

Última atualização: 25/02/2026

1. Dados coletados

O aplicativo não coleta dados financeiros ou informações sensíveis externas. Podem ser inseridos voluntariamente pelo usuário:

  • Idade
  • Sexo
  • Peso
  • Altura
  • Objetivos e preferências de treino
  • Histórico de treinos realizados

Todos os dados são armazenados exclusivamente no dispositivo do usuário.

2. Finalidade

Os dados são utilizados apenas para geração de treinos, estatísticas e organização do histórico.

3. Armazenamento

As informações são armazenadas localmente no dispositivo do usuário.

O aplicativo pode realizar backup opcional dos dados em serviços externos para fins de restauração e continuidade de uso. Esses dados são utilizados exclusivamente para esse propósito e não são compartilhados com terceiros.

4. Notificações

O aplicativo pode utilizar notificações locais para alertas relacionados ao treino. Essas notificações não coletam dados pessoais.

5. Compartilhamento

O Forja não compartilha dados com terceiros, anunciantes ou plataformas externas.

6. Limitação de responsabilidade

O Forja é uma ferramenta de apoio à organização de treinos físicos. As sugestões não substituem acompanhamento profissional de Educação Física.

7. Retenção de Dados

Todos os dados do usuário são armazenados localmente no dispositivo e permanecem enquanto o aplicativo estiver instalado.

O desenvolvedor não armazena, coleta ou mantém dados em servidores externos.

Ao desinstalar o aplicativo, todos os dados são permanentemente removidos.

8. Exclusão de Dados

O usuário pode excluir seus dados a qualquer momento:

  • Limpando os dados do aplicativo nas configurações do dispositivo, ou
  • Desinstalando o aplicativo

Como os dados são armazenados apenas no dispositivo, nenhuma informação é mantida pelo desenvolvedor após a exclusão.
