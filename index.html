Política de Privacidade – Forja

Última atualização: 02/04/2026

1. Dados coletados

O aplicativo não coleta dados financeiros ou informações sensíveis externas. Podem ser inseridos pelo usuário:

  • Idade
  • Sexo
  • Peso
  • Altura
  • Objetivos e preferências de treino
  • Histórico de treinos realizados

Esses dados são utilizados exclusivamente para o funcionamento do aplicativo.

Para garantir a continuidade de uso e recuperação de informações, os dados podem ser armazenados localmente no dispositivo e também enviados para armazenamento externo como parte do sistema de backup do aplicativo.

2. Finalidade

Os dados são utilizados exclusivamente para:

  • Geração de treinos personalizados
  • Exibição de estatísticas
  • Manutenção do histórico do usuário
  • Recuperação de dados por meio de backup

3. Armazenamento

As informações são armazenadas localmente no dispositivo do usuário.

Além disso, o aplicativo realiza backup automático dos dados em serviços externos com a finalidade exclusiva de permitir a restauração das informações.

O desenvolvedor não utiliza esses dados para qualquer outro fim e não os compartilha com terceiros.

4. Notificações

O aplicativo pode utilizar notificações locais para alertas relacionados ao treino. Essas notificações não coletam dados pessoais.

5. Compartilhamento

O Forja não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando necessário para a funcionalidade de backup.

6. Limitação de responsabilidade

O Forja é uma ferramenta de apoio à organização de treinos físicos. As sugestões não substituem acompanhamento profissional de Educação Física.

7. Retenção de Dados

Os dados do usuário são armazenados enquanto o aplicativo estiver em uso.

Os dados armazenados localmente permanecem no dispositivo até que o usuário os remova ou desinstale o aplicativo.

Dados armazenados por meio do sistema de backup podem ser mantidos pelo tempo necessário para garantir a recuperação das informações.

8. Exclusão de Dados

O usuário pode excluir seus dados a qualquer momento:

  • Limpando os dados do aplicativo nas configurações do dispositivo
  • Desinstalando o aplicativo

Após a exclusão, os dados locais são removidos permanentemente.
