Política de Privacidade – Forja

Última atualização: 25/02/2026

1. Dados coletados

O aplicativo não coleta dados financeiros ou informações sensíveis externas. Podem ser inseridos voluntariamente pelo usuário:

  • Idade
  • Sexo
  • Peso
  • Altura
  • Objetivos e preferências de treino
  • Histórico de treinos realizados

Todos os dados são armazenados exclusivamente no dispositivo do usuário.

2. Finalidade

Os dados são utilizados apenas para geração de treinos, estatísticas e organização do histórico.

3. Armazenamento

As informações são armazenadas localmente no dispositivo. Nenhum dado é enviado para servidores externos.

4. Notificações

O aplicativo pode utilizar notificações locais para alertas relacionados ao treino. Essas notificações não coletam dados pessoais.

5. Compartilhamento

O Forja não compartilha dados com terceiros, anunciantes ou plataformas externas.

6. Limitação de responsabilidade

O Forja é uma ferramenta de apoio à organização de treinos físicos. As sugestões não substituem acompanhamento profissional de Educação Física.

7. Data Retention

All user data is stored locally on the user's device and is retained for as long as the application remains installed.

The developer does not store, collect, or retain user data on external servers.

If the user uninstalls the application, all locally stored data is permanently deleted.

8. Data Deletion

Users can delete their data at any time by:

  • Clearing the app data through their device settings, or
  • Uninstalling the application

Since all data is stored locally on the user's device, no user data is retained by the developer after deletion.
